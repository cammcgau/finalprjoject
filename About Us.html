About Us

This is our coding project. We wanted to make this page to reflect on our experiences through the Covid-19 pandemic and how it has encouraged us to code.

Coronavirus and its affects have shifted lots of our perceptions about the way things must be done and the approach that must be taken. Never has lateral thinking been more important. Through this project and learning to code we have been able to pick up a new skill, remotely and create a community with others at the same time.

We hope that you find the page useful, informative and funny.
